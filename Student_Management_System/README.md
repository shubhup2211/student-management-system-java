# Student Management System (Java Console Project)

A console-based Student Management System developed using Core Java and Object-Oriented Programming concepts.

## Features

### Teacher Module
- Teacher Registration
- Teacher Login
- Update Teacher Details
- Delete Teacher
- View Student List
- Enter Student Marks
- View All Student Marks

### Student Module
- Student Registration
- Student Login
- Update Student Details
- View Marksheet
- View Teacher List

## Concepts Used
- Classes & Objects
- Encapsulation
- Constructors
- Arrays of Objects
- Getter & Setter Methods
- Exception Handling
- Loops & Conditional Statements
- Scanner Class
- OOP Principles

## Technologies Used
- Java
- Eclipse IDE

## Project Type
Academic Mini Project / Console Application
